# Embedded Access Control System (Arduino UNO)

## Overview
Password-based door lock system using Arduino UNO with EEPROM memory, servo motor, LCD, LEDs and buzzer.

## Features
- 4-digit password authentication
- EEPROM memory storage
- Servo motor lock/unlock
- LCD status display
- Buzzer alarm on wrong attempts
